C Program to input Digits, Octal and Hex Values into Character Variables

Why This Program is Exceptionally Valuable:
This isn't just a basic input tutorial, it's a bridge from fundamental C concepts to intermediate understanding of number systems. Our program uniquely teaches learners how to input values and simultaneously illustrates core principles of number systems and their direct relationship to character representation via ASCII values in C.

Hands-on Exploration of ASCII Values:
Every character possesses a unique ASCII value. For instance, character 'A' corresponds to 65 decimal, 0101 octal, and 0x41 hexadecimal. This program provides practical experience, demonstrating how you can input 'A' using any of these numerical formats all leading to the identical character output.

Mastering Essential Format Specifiers:
Gain practical real-world knowledge of critical scanf() format specifiers:
%d – Used for decimal (base 10) integer input.
%o – Used for octal (base 8) integer input.
%x – Used for hexadecimal (base 16) integer input.

The Importance of the (int*) Cast:
While the final value might be stored in a char variable (typically 1 byte) scanf() expects an int* (a pointer to an integer) when using format specifiers like %d %o or %x. Omitting this crucial (int*) typecast can lead to compiler warnings or unpredictable program behavior. Therefore applying this cast ensures your code is both safe and functionally clear.
Practical Implementations:


#include <stdio.h>
int main() {
char decChar, octChar, hexChar;

// Accept value in decimal format
printf("Enter decimal value (e.g. 65 for 'A'): ");
scanf("%d", (int*)&decChar);

// Accept value in octal format
printf("Enter an octal value (e.g. 0101 for 'A'): ");
scanf("%o", (int*)&octChar);

// Accept value from hexadecimal format
printf("Enter a hexadecimal value (e.g. 41 for 'A'): ");
scanf("%x", (int*)&hexChar);

// Display characters derived from all formats
printf("\nCharacter from decimal: %c\n", decChar);
printf("Character from octal: %c\n", octChar);
printf("Character from hexadecimal: %c\n", hexChar);
return 0;
}

Output:
Enter decimal value (e.g., 65 for 'A'): 65
Enter octal value (e.g., 0101 for 'A'): 101
Enter a hexadecimal value (e.g., 41 for 'A'): 41
Character from decimal: A
Character from octal: A
Character from hexadecimal: A
