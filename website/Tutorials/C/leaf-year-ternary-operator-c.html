Objective:
The objective of this program is to determine whether a given year is a leap year or not using the ternary operator in the C programming language. Agenda:
The agenda of this program is to guide the user through the process of writing a C program that can check if a given year is a leap year using the ternary operator. By explaining the logic behind leap years and how the ternary operator can be used for conditional expressions, the user will gain a better understanding of how to implement this functionality in their own programs.
Purpose:
The purpose of creating a program to check for leap years using the ternary operator is to showcase the versatility and efficiency of conditional expressions in programming. By utilizing the ternary operator, we can succinctly evaluate conditions and provide a concise output based on the result. This program serves as a practical example of how to implement conditional expressions in C programming, highlighting the simplicity and effectiveness of the ternary operator in decision-making processes.

three numbers using nested if else statements only. */
#include <stdio.h>
int main()
{
 int year;
 // Input year from user
 printf("\n Enter any year:");
 scanf("%d",&year);
 /* check whether the given year is a leap year
or not using ternary operator */
 (year%4==0 &&(year%100!=0||year%400==0))

printf("\n%d is a leap year.",year) :printf("\n %d is not a leap year.",year);

Include the #include<stdio.h> Standard Input-Output Library header file to ensures that the functions for input and output processes are accessible.
int main() function definition starts the execution of the program.
We declare an integer variable year to store the input year. We request the user to enter a year using printf function and read the input using scanf.
The use of ternary operator used in the program above is to find whether the year is a leap year or not.
The condition (year%4==0&&(year%100!=0||year%400==0)) validates whether a year is a leap year.
If the the given condition is true then the printf() function shows that the year is a leap year otherwise in else the printf() shows that the year is not a leap year.
