2. Write a program in C to find the maximum number among given three numbers using nested if else statements only.

Objective:
The objective of this program is to determine the largest number among three input numbers using nested if else statements in the C programming language. By utilizing nested if else statements, we can compare the numbers and identify the maximum value efficiently.
Agenda:
The agenda of this program is to guide the user through the process of writing a C program that can analyse three input numbers and determine which one is the largest. By explaining the logic behind nested if else statements and their role in decision-making, the user will gain a better understanding of how to implement this functionality in their own programs.
Purpose:
The purpose of creating a program to find the maximum number among three given numbers using nested if else statements is to demonstrate the use of conditional statements in programming. By employing nested if else statements, we can compare multiple values and make decisions based on specific conditions. This program serves as a practical example of how to implement decision-making logic in C programming, showcasing the versatility and power of conditional statements.
Real-Life Applications:
In real-life scenarios, the concept of finding the maximum number among multiple values is commonly encountered in various fields such as finance, data analysis, and engineering. For instance, in financial analysis, determining the highest stock price among a set of companies can help investors make informed decisions. Similarly, in data analysis, identifying the largest data point in a dataset can provide valuable insights into trends and patterns. By mastering the skill of finding the maximum number using nested if else statements, programmers can apply this knowledge to solve real-world problems efficiently.
