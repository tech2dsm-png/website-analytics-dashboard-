Master Fahrenheit to Celsius Conversion with This Easy C Program

Fahrenheit to Celsius in C
Got a temperature in Fahrenheit and need it in Celsius? This simple C program does just that! It's a neat little example to see how computers handle basic tasks and calculations.

#include <stdio.h>
int main() {
  float fahrenheit, celsius;
  // Asks you to type in the Fahrenheit temperature.
  printf("Enter temperature in Fahrenheit: ");
  // Waits for your input and saves it.
  scanf("%f", &fahrenheit);
   // This is the core math: Fahrenheit minus 32, times 5, then divided by 9.
  celsius = (fahrenheit - 32) * 5 / 9;
  // Shows you the Celsius temperature, rounded nicely to two decimal places.
  printf("Temperature in Celsius: %.2f°C\n", celsius);
  return 0;
}

Output:
See how It Work (Example)
Run the program, and it'll ask:
Enter temperature in Fahrenheit:
Type 98.6 (like a normal body temp) and hit Enter.
You'll quickly see:
Temperature in Celsius: 37.00°C

Simple as that!
How Each Part Works
Let's break down this temperature conversion program bit by bit:
1. #include <stdio.h>: This line is like grabbing your basic toolbox. It gives your program access to everyday actions like printf() (for putting text on screen) and scanf() (for reading what you type).
2. float fahrenheit, celsius; Here, we're setting up two variables to hold numbers. Since temperatures often have decimal points, we use float to make sure they can store values like 98.6 or 37.00.
3.printf(...) and scanf(...): First, printf() displays the message asking for the Fahrenheit temperature. Then, scanf() listens for your number, grabs it and puts it into the fahrenheit . The & simply tells scanf() where to store that number.
4. celsius = (fahrenheit - 32) * 5 / 9;: This is the main calculation. It takes the Fahrenheit number, does the standard conversion math and puts the new Celsius value into the celsius spot.
5.printf("Temperature... %.2f°C\n", celsius);: Finally, printf() shows your answer. The %.2f bit is just a neat trick to make sure the number always shows with exactly two decimal places, keeping your output clean.
