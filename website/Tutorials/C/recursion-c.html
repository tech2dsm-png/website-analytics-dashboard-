Recursion Function in C Language

In this tutorial we will learn Recursion Function in C programming Language.
Recursion Function: The recursion is the process in c programming language which allows to repeat the certain section of code in a program.
Thus, in the C programming languages the program allows the user to call the self function inside the same function, it is known as a recursive call in that function .
when ever the function calls same copy of itself then it is the actual recursion . It is useful when we do want to work on a smaller problem.
All the problems can not be solved by using recursion in C programming Langguage. It is mainly useful and benificial for the similar types of subtask.For e.g. one can apply recursion on sorting, searching, and traversal problems.
But one important thing we need to keep in our mind that the iterative or repeatative(looping) solutions are very efficient than the recursion. why It is efficient because the function call is almost overhead all the time.
The recursion in a C program continues further until and unless it meets some condition to ultimately prevent it. The recursion function continues its process infinite times. To stop or prevent the continues execution of the function we should mention the exit condition.
Thus, if we want to prevent recursion in our programming code we have to use the if…else statement or some similar type of acondition or approch where one of the branches will makes a recursive call while the other one doesn’t.

  Working of Recursion Function in C Programming Language.
In C programming language the recursion can be made by calling function to itself. The process of calling function to itself is called recursion.
The recursive function performs some specific task by dividing it in to subtask.
Let understand the recursion function from the given diagam.
we have declared and defined the function recur_fun().
The main() function calls the recur_fun(). The control transfers to the recur_fun() and start executing statements inside recur_fun() body.
Inside recur_fun() we have again called the recur_fun() which calls the same function again and this process continues infinite times.
There should be a termination condition that we need to define in the function to stop infinite execution of recursion function. Some specific subtasks should satisfy this termination condition. Due to this, the recursive execution of function stops and we get the final result.

C program to find Factorial of a Number .

The following example given is a simple implementation of recursion functions in C Programming that calculates the factorial of a number.

#include <stdio.h>
int fact_n(int n)
{
  if(n==1 || n==0)
  {
   return 1;
   }
  else
  {
    return n*fact_n(n-1);
  }
}
int main()
{
 int x;
 printf("\n Enter any Number");
 scanf("%d",&x);
 printf("\n The Factorial of %d!=%d",x,fact_n(x));
 return(0);
}
