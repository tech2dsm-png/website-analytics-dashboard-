Electricity Bill Calculation in C Using Switch Case

Objective:
The target of this program is to work out the power bill in light of the quantity of units utilized by the client. The program will utilize the switch case statement to ascertain the unit cost and compute the all out bill.
Agenda:
1. Request that the client enter the quantity of units utilized.
2. Utilize the change case determination articulation to set the unit rate in light of the quantity of units utilized.
3. Compute your all out bill by duplicating the unit rate by the quantity of units utilized.
4. Show the all out charge add up to the client.
Purpose:
This program means to give clients a straightforward and proficient method for computing power bills in view of their use. By utilizing switch case statement, the program can rapidly decide the unit rate and ascertain the complete bill sum without complex estimations.
Real-Life Applications:
electricity bills are a typical cost for families and organizations all over the planet. By having a program that can rapidly and precisely compute bill sums in light of the quantity of units utilized, clients can deal with their electricity use and financial plan successfully. This program can be involved by organizations to create exact solicitations for their clients, as well as by people to compute their costs every month. /* a program in C to calculate the electricity bill based on units consumed using switch case. */
#include <stdio.h>
int main()
{
 int units;
 float totBill=0;
 printf("\nEnter the number of units consumed:");
 scanf("%d",&units);
 // Calculate electricity bill based on units consumed
 switch(units/100)
 {
 case 0:
 totBill=units*1.20;
 break;
 case 1:
 totBill=100*1.20 + (units - 100)*2.00;
 break;
 case 2:
 totBill= 100*1.20 + 100*2.00 + (units - 200) * 3.00;
 break;
 default:
 totBill= 100*1.20 + 100*2.00 + 100*3.00 + (units - 300)*3.50;
 break;
}
 printf("\nTotal electricity bill: $%.2f\n",totBill);
 return 0;
}

Explanation
The program first asks the user to enter the number of units used. It then uses a switch statement to calculate the total electricity bill based on the number of units used.
Tariffs are as follows:
The rate for the first 100 units is $1.20 per unit.
For the next 100 units the rate is $2.00 per unit.
For the next 100 units, the rate is $3.00 per unit.
Each additional unit costs $3.50 per unit.
The total bill is calculated based on the rates and displayed to the user.
Finally, return 0 indicates successful execution of the program.
