Q1.Who is the father of C language?
a) Dennis Ritchie
b) James Gosling
c) Steve Jobs
d) Rasmus Lerdorf
Answer: a
Dennis Ritchie is the father/founder of the C programming language. The C programming language was developed in 1972 by the American Telephone and Telegraph Bell Laboratories in the USA.

Q17.What is the range for integer for 16-bit?
a) -32768 to 32767
b) 3.4e38 to 3.4e38
c) -32767 to 32768
d) -32668 to 32667
Answer: A
Explanation: In a 16 Bit C compiler we have 2 bytes to store an integer, and 1 byte for a character. For unsigned integers the range is 0 to 65535. For signed integers the range is -32768 to 32767. For unsigned character, 0 to 255
