12.Write a program in c to check if a given number is a strong number or not using a while loop.

Strong Numbers
A strong number is a number that is the sum of the factorial of its individual numbers (digits).
Examples
Let's say the number is 145, which equals 1! +4! +5! = 1 + 24 + 120 = 145.
145 is a strong number because the sum of its multiples equals the number itself.
Let's say the number is 112, which equals 1! +1 +2 = 1 + 1 + 2 = 4 Here 4 does not equal 112.
Therefore, 112 is not a strong number.
Description
Write a C program that prompts the user for a number and then checks whether the number is a strong number. Additionally, create a program that examines for strong numbers within a certain range.

Problem Solution
1.Take a number as input and store it in a variable called number.
2.Extract the digits of a number and find the factorial of each digit number.
3.Calculate the sum of the factorial of each digit number.
4.If the sum equals a number, print Number is a strong number.
5.If the sum does not equal the number, print The number is not a strong number.

Check whether a given Number is strong Number using while loop in C

/*
* Check whether a given Number is strong Number using while loop in C
*/
#include<stdio.h>
int factorial(int n)
{
 int i, fact = 1;
 for (i = 1; i <= n; i++)
 {
    fact=fact*i;
 }
 return fact;
}
int main(void)
{
 int number, sum = 0;
 printf("Enter the number: ");
 scanf("%d",&number);
 int i=n;
 while (i!=0)
 {
   sum = sum + factorial(i%10);
   i = i / 10;
 }
 if (sum == n)
 {
   printf("\nThe number %d is strong number",number);
 }
else
 {
    printf("\nThe number %d is not a strong number", number);
 }
return(0);
}

Output
Output:
Enter the number: 145
145 is a strong number
Enter the number: 112
112 is not a strong number

Explanation
1.In this C program, we read the exact value using the variable "number".
2.Find the digit of number and the factorial of each digit.
3.Calculate the factorial sum of each digit.
4.If the sum is equal to the number, then print "The number is a strong number".
5.If the sum is not equal to the number, then print "The number is not a strong number".
